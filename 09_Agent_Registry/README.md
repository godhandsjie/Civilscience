# 09_Agent_Registry

Registry of AI agents and their capabilities.

## Purpose

This folder records AI agents built on top of:

- Skills
- Prompts
- Workflows
- Tools

## Agent Categories

Future agents may include:

- Research Agent
- Coding Agent
- Investment Agent
- Business Agent
- MR Cultural Technology Agent

## Agent Definition

Each agent should describe:

- Mission
- Responsibilities
- Required skills
- Available workflows
- Input format
- Output format
- Constraints
