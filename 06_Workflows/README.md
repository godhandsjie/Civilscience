# 06_Workflows

Standard operating procedures for AI-assisted work.

## Purpose

This folder stores repeatable workflows combining:

- Skills
- Prompts
- Tools
- Agents

## Categories

- Research workflow
- Software development workflow
- Business workflow
- Investment workflow

## Workflow Structure

Each workflow should define:

- Goal
- Input
- Required skills
- Execution steps
- Output format
- Quality checklist
