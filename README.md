# Civilscience

## Personal AI Operating System

Civilscience is a structured AI knowledge and capability system designed for:

- Scientific research
- Software development
- Business analysis
- Creative innovation

The goal is to transform AI usage from isolated conversations into a continuously improving personal AI infrastructure.

---

# Architecture

Civilscience follows a layered AI operating system architecture:
System
  ↓
Skills
  ↓
Prompts
  ↓
Workflows
  ↓
Templates
  ↓
Agents
  ↓
Projects

---

# Repository Structure

## 00_System

Core rules and architecture.

Contains:

- AI operating principles
- Agent design standards
- Naming conventions
- Management rules


## 01_OpenSource_Skills

Public reusable AI skills.

Examples:

- Scientific research
- Literature analysis
- Data processing
- Coding workflows


## 05_Prompts

Reusable prompt library.

Categories:

- Research
- Coding
- Writing
- Investment
- Business


## 06_Workflows

Repeatable AI workflows.

Combines:

- Skills
- Prompts
- Tools
- Agents


## 07_Templates

Standard templates for creating:

- Skills
- Agents
- Projects
- Documentation


## 09_Agent_Registry

Registry of AI agents.

Defines:

- Agent missions
- Capabilities
- Required skills
- Associated workflows

# Design Philosophy

AI capability should be accumulated as structured knowledge.

The objective is not only to use AI tools, but to build a scalable personal AI ecosystem.
